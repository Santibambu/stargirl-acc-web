# Stargirl Acc. – Sitio web de exhibición

Sitio web desarrollado durante un proyecto colaborativo de la secundaria, orientado a la exhibición de productos de un emprendimiento.

## Contexto del proyecto
Este sitio web formó parte de un proyecto interdisciplinario entre estudiantes de las orientaciones de **Comunicación** y **Administración** de quinto año del Instituto Medalla Milagrosa.  
El objetivo general del proyecto fue el desarrollo y promoción de un emprendimiento real, que incluyó distintas contribuciones como publicidad, diseño gráfico, piezas sonoras y estrategias de comunicación.

Dentro de ese marco, este sitio web fue **una contribución individual**, desarrollada de forma autónoma, con el objetivo de exhibir los productos del emprendimiento.

## Rol personal
- Diseño y desarrollo completo del sitio web  
- Estructuración de secciones y navegación  
- Adaptación del contenido al enfoque del emprendimiento  

## Tecnologías
- HTML
- CSS

## Referencia de diseño
La estructura y estética del sitio están basadas en el proyecto **Plantitas Hermosas**, otro trabajo académico desarrollado previamente, reutilizando y adaptando la organización general del sitio para un nuevo contexto.
